const UBasicStringView< CharT, Traits >  str 
)
inline

Behaves as a FormattedOutputFunction.

After constructing and checking the sentry object, determines the output format padding as follows:

  • If v.size() is not less than os.width(), uses the range [v.begin(), v.end()) as-is.
  • Otherwise, if (os.flags() & ios_base::adjustfield) == ios_base::left, places os.width()-v.size() copies of the os.fill() character after the character sequence.
  • Otherwise, places os.width()-v.size() copies of the os.fill() character before the character sequence.

Then stores each character from the resulting sequence (the contents of v plus padding) to the output stream os as if by calling os.rdbuf()->sputn(seq, n), where n=std::max(os.width(), str.size()). Finally, calls os.width(0) to cancel the effects of std::setw, if any.

Exceptions
Maythrow std::ios_base::failure if an exception is thrown during output.
