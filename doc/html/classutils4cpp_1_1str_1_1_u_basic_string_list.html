Returns the index position of the first occurrence of str in the list, searching forward from index position from.

Returns -1 if no item matched.

Example:

BasicStringList<std::string> list;
list << "A" << "B" << "C" << "B" << "A";
list.indexOf("B"); // returns 1
list.indexOf("B", 1); // returns 1
list.indexOf("B", 2); // returns 3
list.indexOf("X"); // returns -1

If cs is CaseSensitive (the default), the string comparison is case sensitive; otherwise the comparison is case insensitive.
