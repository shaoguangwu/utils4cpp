Detailed Description

template<class StringT>
class utils4cpp::filesystem::UPath< StringT >

Objects of type path represent paths on a filesystem.

Since
v0.0

Only syntactic aspects of paths are handled: the pathname may represent a non-existing path or even one that is not allowed to exist on the current file system or OS.
